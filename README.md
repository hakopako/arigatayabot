# IRC-bot
app store のありがたいレビューを流してくれるbot   

## require
Node.js  
CoffeeScript `npm install -g coffee-script`  
Forever `npm install -g forever`  

## usage 
```bash
cd arigatayabot
cp run.sh.example run.sh
# ---- edit run.sh -----
sh run.sh
```
